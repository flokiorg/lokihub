
This release brings full global language support to Lokihub, alongside a suite of UI and UX improvements designed to deliver a more seamless and intuitive experience.

## Major Changes

###  Internationalization (i18n)
- **Multi-language Support:** Lokihub now supports 7 languages:
  - English (EN)
  - Spanish (ES)
  - Japanese (JA)
  - Korean (KO)
  - Chinese (ZH)
  - Arabic (AR)
  - Farsi/Persian (FA)

- **RTL Support:** Full support for Right-to-Left (RTL) layouts for Arabic and Farsi, including mirrored sidebars, navigation, and directional icons.
- **Local Fonts:** Bundled `Noto Sans Arabic` and `Vazirmatn` fonts to ensure high-quality rendering for RTL languages without external dependencies.
- **Language Switcher:** New components to easily toggle between supported languages.

###  Lightning & Wallet
- **JIT Payment Explanation:** Added a new educational dialog for Just-In-Time (JIT) payments to help users understand how liquidity is managed.
- **Dynamic Formatting:** Improved currency input formatting with automatic thousands separators for better readability.
- **App Presets:** Implemented unique FLC/Loki presets tailored for individual apps.

###  UI/UX Improvements
- **Loading State:** Refactored the global Loading component for better accessibility and theme consistency.
- **Layout Consistency:** Standardized the use of `AppHeader` across more screens, including inbound channel management.
- **Visual Polish:** Aligned inbound channel order limits with outgoing balance text for a cleaner look.

## Technical Notes
- Integrated `react-i18next` for robust translation management.
- Implemented a direction-aware CSS engine for seamless RTL/LTR switching.
