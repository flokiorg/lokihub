
This release brings background running support, a system tray icon, improved reliability, and a more polished user experience.

## What's New

### 🛠 System Tray & Background Running
- **Runs in the background:** Lokihub stays active after you close the main window. You can fully quit the app at any time from the tray icon menu.
- **System tray icon:** A tray icon is now available on macOS, Linux, and Windows, giving you quick access to the app without keeping the window open.
- **Cleaner dock behavior (macOS):** The app icon in the dock appears and disappears automatically based on whether the window is open, so your dock stays tidy.

### ✅ Reliability & Stability
- **Fewer crashes:** The app now handles unexpected errors in isolated areas without taking down the entire interface.
- **Better payment attribution:** Payments can now be more accurately linked to the app or service that generated them.
- **Improved internal stability:** Significant testing work was done across payment processing and exchange services to ensure more reliable behavior.

### 🎨 Interface & Experience
- **New Help section:** A dedicated Help and FAQ section is now available to guide you through common tasks and troubleshooting.
- **More languages supported:** Help content is now available in additional languages.
- **Visual polish:** Navigation, headers, and transaction lists have been refined for better consistency and responsiveness across all screen sizes.
